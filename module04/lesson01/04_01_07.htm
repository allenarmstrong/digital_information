Fast, Easy, Excel-lent

How awesome would it be if your wallet automatically refilled with money? Unfortunately, we don't have a button for that. But Excel provides many features to expedite creating and enhancing a spreadsheet.

AutoComplete

Excel's AutoComplete feature helps make your data entry faster and easier. As you type data into a cell, if the first few characters match an existing entry in that column, Excel automatically enters the remaining characters for you (if the entry is text or a combination of text and numbers).

Hyperlinks

Talk about automatic! A hyperlink automatically connects text on your document to another page, website, email, picture, etc.

Once he was done, he shared his wish list electronically with his friends and family. Have we got your wheels turning, yet? If you would like to insert a hyperlink, follow these directions:

Directions

  1. Select the cell with the text you want to link.
  2. Go to the Insert Tab and select "Hyperlink."
  3. Do one of the following:
    1. Add a URL (address of a website).
    2. Add a location in the document (specify by using the column number followed by row letter, i.e. A3, B9, CL56).
    3. Add an email address.
  4. Select "OK.”
