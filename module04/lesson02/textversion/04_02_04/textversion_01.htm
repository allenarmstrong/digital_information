Functions—Text Version

Functions can be used to calculate information in cells.

To add data from joining cells, use the SUM function. Type "equal SUM" with an open parenthesis, then select the cells to add together. Complete the function by adding an end parenthesis, and hit "Enter."

Since a relative cell reference was used, the formula will continue to sum up the data in corresponding cells when you fill down.
