Spreadsheet Rows and Columns The spreadsheet has five columns and six rows, and it is formatted as a table. The spreadsheet is missing some or all of the columns and/or rows, and it is not formatted as a table.  
Headings The spreadsheet has the appropriate headings for each column and row. The spreadsheet does not have the appropriate headings for each column and/or row.  
Hyperlink The heading "Menu Item" is hyperlinked to the selected restaurant's nutritional data. The heading "Menu Item" is not hyperlinked to the selected restaurant's nutritional data.  
Menu Items The spreadsheet has an entrée, side, dessert, and drink. The spreadsheet is missing an entrée, side, dessert, and/or drink.  
Nutritional Data The nutritional data is entered in the proper cell for each menu item. The nutritional data is missing in some or all menu item cells.  
Calculations Calculations have been performed correctly, using the proper Excel functions. Calculations have not been performed correctly and/or have not used the proper Excel functions.  
Chart The chart compares the fat calories and non-fat calories for each menu item. The chart does not compare the fat calories and non-fat calories for each menu item.  
Chart/Vertical Axis Title The chart is titled "Fat Calories and Non-Fat Calories in Meal," and the vertical axis is titled "Calories.” The chart is not titled "Fat Calories and Non-Fat Calories in Meal," and/or the vertical axis is not titled "Calories.”  
Analysis C6 is highlighted appropriately, and C7 is labeled properly based on the IF function. C6 is not highlighted appropriately, and /or C7 is not labeled properly based on the IF function.
