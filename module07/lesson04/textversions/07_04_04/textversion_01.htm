Animations Tab—Text Version

Animation allows you to add a motion effect to objects like bullet points, graphics, titles, text, etc. when they enter the slide.

Existing only appears active when you upload a presentation created using the desktop version of PowerPoint. Existing allows you to keep the animations used on the desktop version even though they may not be offered on the PowerPoint Online App.

None allows you to remove a animation from a slide.

Appear makes the animated object suddenly appear on the slide.

Fade In makes the animated object fade in on the slide.

Fly In makes the animated object fly in on the slide. Fly In has four Effect Options to signal the direction from where the object will fly in: From Bottom (object fly in from bottom), From Left (object fly in from the left), From Right (object fly in from the right), and From Top (object fly in from top).

Effect Options allows you to choose special effects for the way an animated object enters the slide.

Timing allows you to reorder the animated object in two ways: Move Earlier (the selected object will enter the slide earlier than another object on the slide) or Move Later (the selected object will enter the slide later than another object on the slide).
