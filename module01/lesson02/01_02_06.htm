A File Type for Every Conceivable Task

At all the different levels in a computer's file storage you can find different file formats, and some of them might be the kinds you see in this chart. A file format is a way that data is stored as a file. File formats are linked to certain programs. Jordan, with all his experience, knows almost instinctively that a file with the XLS extension is used with a spreadsheet program. Some file formats are not only specific to the program that uses them, but the operating system they are used on.

Every file has one or more programs that can open that kind of file so you can do things with the file, such as change it or view it. The chart here shows many common file types, but there are many more. You will discover and use some of them as your IT skills increase, and others you may never see more than a few times. There are so many file formats out there, but you really only need to know the ones you use most often.
