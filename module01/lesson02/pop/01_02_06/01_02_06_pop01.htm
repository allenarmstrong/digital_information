Common File Types and Extensions

Document Files

  • DOC: An editable document format created by Microsoft and used in its Word program.
  • PDF: Portable Document Format is a proprietary document file type created by Adobe Systems that is compatible with most computer systems. Adobe's Acrobat Reader is the most common application to view these files.
  • XLS: A type of spreadsheet file created by Microsoft and used in its Excel program.

Image Files

  • GIF: Graphics Interchange Format uses compression so these files can be viewed on the Internet, especially for slow connections. Web browsers can view these files.
  • JPG: created by the Joint Photographic Experts Group (JPEG), this "lossy" type compresses image files for use one on the Internet. Web browsers can view these files.
  • PS: A graphics programming file type used for desktop publishing and printing programs directly.

Video Files

  • AVI: Short for Audio/Video Interlaced, this is a file format used for video. Use a program like Windows Media Player for this file.
  • MPEG: A video file type created by the Moving Picture Experts Group (MPEG) that is well-suited for use over the Internet. Windows Media Player can view these files.

Audio Files

  • MP3: A common audio file format that uses "lossy" compression to reduce the size of sound or music files so they can be used on devices like MP3 players or played in media programs like Windows Media Player, or iTunes.
  • OGG: An open source audio file type that can be played by compatible software audio players and hardware devices.
