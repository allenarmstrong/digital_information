File and Folder Security Must Be a Habit


Jordan has saved the world, well at least this part of the world. He organized the computer so that the files on the desktop are now in separate folders labeled by file type. GIFs and JPGs are in an "images" folder, while DOC and PDF files are in a "documents" folder. After putting in the new hard drive, Jordan moved some files onto it, so that the computer's operating system had more of its own file space to use. He put control over the file naming in the hands of the service member at the hidden, top-secret site. While others will still use this computer, the service member now understands the importance of good file maintenance. Jordan's work here means that the data is more organized: the computer is secure. He has done his part to keep the nation's top-secret documents out of the hands of the enemy.
