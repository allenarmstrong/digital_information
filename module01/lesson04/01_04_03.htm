What Type of Network Do I Need?

Should you use a client/server or P2P network? It all depends on what you need. See below for some benefits and risks for each type of network:

IT professional maintains who is added and removed from the network.

Requires an IT professional, but easy to add clients to server and give privileges.

More expensive; cost of server and server maintenance.

Important files and programs are accessed and stored on main server. Data is automatically backed up and maintained in a safe environment.

Clients depend on a working main server to perform their jobs.

Security levels are set up by an IT professional to ensure a safe network.

Servers typically have plenty of RAM and operating software that increases the speed they at which process information.

There is a need to maintain an IT department in order to ensure the server and the computers connected to it operate properly.

Software is maintained on the server, allowing for easy access to upgrade programs for all users.

Individuals can easily add or remove devices from their network.

No IT professional required to maintain server; users are responsible for managing their own computers.

Less expensive; no server cost.

Files and programs are accessed and stored on individual computers. Data is backed up by each individuals when and how they choose

Clients work independently; therefore, if one computer is not working, other computers can still do their jobs.

Security levels are set up by individuals; therefore, these networks may not be as secure.

Computers typically do not have the same amount of RAM as a server; therefore, they will operate more slowly when someone is accessing content on their computer.

There is no need for a big IT department to keep the network up and running.

Software is maintained on each computer, requiring each client to upgrade programs.
