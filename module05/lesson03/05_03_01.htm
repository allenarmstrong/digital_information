Accepting Applications

Without web applications, you couldn't follow your favorite celebrity on Instagram, or post a selfie to Facebook, or watch a Vine video. However, there is so much more than following people and watching movies. In fact, if you can think it, there is probably a web or mobile app out there that can do it. (And if not, you could make big bucks creating it!)



After completing this lesson, you will be able to:

  • define web-based and mobile applications
  • describe how to use web and mobile applications
  • describe appropriate use of social networking sites, blogs, and collaborative tools
