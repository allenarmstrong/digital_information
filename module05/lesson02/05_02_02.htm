Ooey, GUI Basics

You are most likely using a GUI (pronounced "goo-ey") right now to read this page! For example, each time you double click, move a pointer around a page, or tap a screen, you are using GUI to communicate with your device. Microsoft Windows and Apple Macintosh use a WIMP interface to allow users to select objects, issue commands, and run programs. Here are the elements of a WIMP interface:

  • Windows are what you see when you run self-contained programs, like a browser, a word-processing document, an Excel spreadsheet, email, etc.
  • Icons are small images that represent a program, function, app, etc. and act as a shortcut.
  • Menus allow users to choose a command, location, program, or task from a list.
  • The pointer is an onscreen symbol (arrow or hand) that users control to select icons, data elements, tools, apps, etc.

Tablets, smartphones, computer kiosks, and some desktop monitors use touch screen interfaces to communicate.
