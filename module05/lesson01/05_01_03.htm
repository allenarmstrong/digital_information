Make a Connection

In today's wireless era, it's easy to take connecting to the internet for granted, meaning we typically tap a button and have instant access. But it's important to know what's necessary to make this happen. There are three required elements to connect to the internet:

  • Internet Service Provider (ISP): dial-up, DSL, cable, satellite, 3G and 4G (wireless).
  • Hardware: modem, router, or network card (wireless).
  • Web Browser: Windows defaults to Internet Explorer, Apple defaults to Safari, and Android defaults to Chrome.

Once you connect to the internet, you can use your web browser to navigate the World Wide Web.

Okay, so it's not magic. it's the browser using behind-the-scenes elements of the URL to access the information you requested:

  • Internet Protocol (IP): "https"
  • Domain Name Server: "www.hourofcode.com"
  • Internet File Name: "web-server.htm"

In a nutshell, the browser passes the domain name to a domain name server that maps it to the correct IP address. Then, the code behind the page is translated from HTML to a human-readable format, and you enjoy the web page you requested.
