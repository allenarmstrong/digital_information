Where Are You Going?


Unlike "LOL" and "IDK," URL is not an abbreviation that most people know off the top of their heads. It stands for Uniform Resource Locator, and it's how the internet knows where to take you when you type an address into your browser.

A URL is a formatted string of text that web browsers, email applications, and other software programs use to identify a particular resource on the internet. Most URLs are located in the address bar of a web browser and consist of three separate parts (called substrings):

  • Network Protocol – A system of rules that defines how internet resources get communicated on a network
  • Host – The computer or other network device where the resource is located
  • Resource Location – The path to a particular network resource on the host
