Table Formatting—Text Version

Now that you have a basic table, let's dress it up a little. You will notice the Design and Layout ribbons have now appeared.

To change the design of your table, go to the Design ribbon and select a table style.

Next, make your tasks stand out by highlighting each task in a specific color. Choose the cell you want to highlight, in this case "Dishes," select "Cell Shading," and the color yellow. Repeat this step for all cells that contain the word "Dishes."

We'll do the same cell shading with the "Trash" task. This time we'll highlight in blue.

Next, use these steps to highlight each task in a different color. This makes your table easy to read at a glance.
