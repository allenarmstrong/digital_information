02.03 Word Basics Rubric

(30 points possible)

Criteria Excellent
3 points		 Needs Improvement
2–0 points		 Score Yourself
Text The text is typed correctly, and the student's name and date appear in the proper places. The text is not typed correctly, and/or the student's name and date do not appear in the proper places.  
Numbered List The list is properly numbered 1–3, and each item describes the student. The list is not properly numbered 1–3, and/or each item does not describe the student.  
Font Style and Size The font style is Times New Roman, and the font size is 12 point. The font style is not Times New Roman, and/or the font size is not 12 point.  
Font Color The font color of the whole document is blue. The font color of the whole document is not blue.  
Bold The student's name and date are in bold font. The student's name and/or date are not in bold font.  
Italicize The word "excited" is in italics. The word "excited" is not in italics.  
Underline The word "free" is underlined. The word "free" is not underlined.  
Highlight The 4th sentence is highlighted in yellow. The 4th sentence is not highlighted in yellow.  
Indent The first line of each paragraph is indented. The first line of each paragraph is not indented.  
Save File The document has been saved with the proper file name. The document has not been saved with the proper file name.
