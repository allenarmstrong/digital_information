Creating a List

Creating a numbered list in Word is as easy as 1, 2, 3. (Pun intended.) Word Online has features to create numbered or bulletted lists that automatically number or bullet each new line. You are going to create a list of your top five TV shows, books, songs, games, or restaurants.

Watch the Creating a List video to learn to create a numbered list and practice your formatting skills:


Next, you're going to create a bulleted list. Include your top five TV shows, books, songs, games, or restaurants; just make sure you select a different category than you used for the numbered list. Also, follow these directions to show your formatting skills.

The bulleted list should be font style "Times New Roman" and font size "18." Also underline the first bullet, use blue font for the second bullet, bold the third bullet, italicize the fourth bullet, and highlight the fifth bullet in green. See the example below:

  • First bullet—Underline
  • Second bullet—Blue Font
  • Third bullet—Bold
  • Fourth bullet—Italic
  • Fifth bullet—Highlight in green

Save the file to your OneDrive using this file name format: LastNameFirstName_Lists. For example, if your name is John Smith, you would save the file as "SmithJohn_Lists."
