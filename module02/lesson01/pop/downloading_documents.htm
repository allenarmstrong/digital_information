Downloading Documents

Use the following directions to download individual documents or zip multiple documents into one folder.

Downloading Documents from OneDrive


  1. Open your OneDrive to locate your documents.
  2. Select the document you want to download. (A checkmark will appear to indicate the document has been selected.) Select "Download."
  3. Select "Save."
  4. Select "Save As."
  5. Select "Documents" to specify where to save your document, and then select "Save."
  6. Your document is now saved to your Documents Library on your computer.

Zipping Files

  1. Open your OneDrive to locate the folder you want to zip.
  2. Select the folder you want to download. (A checkmark will appear to indicate the document has been selected.) Select "Download."
  3. Select "Save."
  4. Select "Save As."
  5. Select "Documents" to specify where to save your folder, and change the "Save as Type" to "Compressed (zipped) Folder (.zip)" before selecting "Save." 
  6. Your document is now zipped.
Mac Users: Your folder will automatically unzip when you zip a folder from your OneDrive. Follow these steps to re-zip your folder:
  1. Locate the items to zip in the Mac Finder (file system).
  2. Right-click on a file, folder, or files you want to zip.
  3. Select "Compress Items."
  4. Find the newly created .zip archive in the same directory.
