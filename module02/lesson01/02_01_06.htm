Take an Office Tour

It's great to be able to explain the parts and features of Office Online, but to maximize your PACE, you need to be able to find and use the productivity tools. So let's tour Office Online.

Signing In

Signing In

To get to your new "Office," you must first sign in. Follow the steps below to learn how:

  1. Go to Office.com.
    Important: You will need to create a free Microsoft Office Online Account. Watch the How to Create an Office Account video for directions.
  2. Sign in with your username and password.
  3. Once you're on the home page, select the icon for Word Online, Outlook, People, OneNote, PowerPoint, Calendar, Excel Online, or OneDrive to open the program.
Creating New Folders

Creating New Folders

To make the most of your online workplace, it's important to be organized. One way to do this is to create folders to house your documents.

Uploading Existing Documents

Uploading Existing Documents

Before cloud-based programs existed, files and folders were saved to a computer's hard-drive or an external device like a flash-drive. If you have existing documents, you can upload these to your OneDrive, so you can store, access, and edit them from any device as long as you have internet access.

File and Folder Properties

File and Folder Properties

The cloud-based nature of the Office Online tools makes sharing your work and collaborating with others a breeze. To do this, you can adjust a file or folder's properties to allow some people to view your work, while allowing others to view and edit it.


  1. Select the arrow next to the folder or file you want to share.
  2. Select the "Share" button at the top of the OneDrive.
  3. Type the email address of the people you want to share the folder or file with.
  4. Select the permission for the recipient.
    1. You can allow recipients to edit the folder or file by selecting "Can edit."
    2. If you only want the recipient to be able to view the work but not make any changes, select "Can view."
  5. You can add an optional note in the message box.
  6. Select "Share" when you are ready to send the folder or file to the recipient. Voila! Collaboration in action.
