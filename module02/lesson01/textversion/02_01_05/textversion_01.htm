Creating New Folders—Text Version

Let's get organized. We'll start by creating folders in OneDrive so you can easily find your files.

To create a folder, select "New" and then "Folder."

Give the folder an appropriate name and select "Create." You should now see your new folder appear in your OneDrive.

You can make as many folders as you want, but remember to keep files organized so they are easy to find when you need them.
