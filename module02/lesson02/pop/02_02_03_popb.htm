How to Add Attachments—Text Version

Time to attach a file to your email.

Once you've created an email, select "Insert." Here, you can insert a file three different ways. For now, we'll insert using "Files as attachments."

Find the file you want to attach to your email on your computer, and select "Open."

The attachment is now in your email and ready to send.
