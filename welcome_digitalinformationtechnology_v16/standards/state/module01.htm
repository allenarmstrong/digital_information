State Standards: Module One

State Standards by Lesson
Lesson Standard
01.01 Computer Parts and Pieces
  • 4.11 Explain the history and purpose of various operating systems. (e.g., DOS, Windows, Mac, and Unix/Linux)
  • 5.01 Explain the general architecture of a microcomputer system.
  • 5.02 Explain the need for and use of peripherals.
  • 5.03 Demonstrate proficiency using peripherals.
01.02 The Secret Life of Files
  • 4.03 Identify and describe types of file systems and classify common file extensions based on software application programs used in the workplace environment.
  • 4.05 Demonstrate basic computer file management skills and file naming conventions to accurately organize files into hierarchies by labeling file folders for easy accessibility.
  • 4.08 Identify operating system file naming conventions.
  • 4.10 Demonstrate a working knowledge of standard file formats.
  • 4.09 Demonstrate proficiency with file management and structure. (e.g., folder creation file creation, backup copy, delete, open, save)
01.03 Computer Upkeep
  • 4.06 Discuss the process of troubleshooting problems with computer hardware peripherals, including input and output devices in the workplace environment.
  • 5.04 Identify the basic concepts of computer maintenance and upgrades.
  • 5.05 Differentiate between diagnosing and troubleshooting.
01.04 Networks and What They Do
  • 6.02 Identify the role of servers and clients on a network.
  • 6.03 Identify benefits and risks of networked computing.
  • 6.04 Identify the relationship between computer networks and other communications networks. (e.g., wifi, teleconference, telepresence)
  • 6.06 Demonstrate basic understanding of network administration.
