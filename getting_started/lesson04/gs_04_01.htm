Course Specifics: What You Need To Know

Materials

Students will need to create an Office.com account. There is no cost associated with creating this account and using the Office Online applications. Students will use different Office Online applications to complete assignments within the course. All teaching around the use of word processing, spreadsheets, and presentation software is done using these applications. The online version of these applications has different features than the desktop version; therefore, use the online applications when following instructions and completing assignments.
