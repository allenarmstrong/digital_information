Types of Assessments (continued)

Collaboration

You are required to complete one collaboration activity per segment. This is an opportunity to work with one or more of your classmates and have deeper conversations concerning class issues and topics. Select the buttons below to learn more about collaboration.

Synchronous Collaboration

In some cases, synchronous collaboration opportunities will be provided. The opportunities will allow you to interact with your classmates online, in real-time, or by using Web 2.0 tools.

Asynchronous Collaboration

You may also have the opportunity to participate in an asynchronous collaboration with one of your classmates. Instructions on how to participate are provided in the course.

Tools for Assessments

Research Note-taking Template

A note-taking template can help you prepare for and organize your research by doing these things:

  • Record information by paraphrasing, summarizing, or using direct quotes from the text
  • Collect available source information like author name, date of access, date of publication, page number, title, etc.
  • Process your thoughts about the information you're reading

Rubrics

Grading rubrics are provided for all project and essay-based assignments.

Zipping Files

This method must be used when submitting multiple files within an assignment. Simply follow these steps:

  1. Select the folder that contains the files.
  2. Right-click on the folder that contains the files and select Send To > Compress (zipped) folder.
    When you do this your files inside this folder will be converted into a compressed file, which looks like a folder with a zipper on it.
  3. Submit the zipped folder in the assessments area.
