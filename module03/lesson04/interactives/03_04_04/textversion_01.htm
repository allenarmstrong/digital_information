S.M.A.R.T.—Text Version

S: Scope. The project manager must determine the overall work that needs to be completed to deliver the desired product, service, or end result. If the scope is not accurate, it could negatively impact the cost and timeline of a project.

M: Money. Once project managers estimate the scope of a project, they must figure out the budget (estimated amount of money needed to complete a project) to deliver a quality product. Time is money, so project managers need to enlist the best people and resources to get the job done quickly.

A: Analysis. Project managers must analyze the possible risks of a project like inaccurate scope, budget shortfalls, insufficient resources, missed deadlines, etc. Frequent meetings with team members can help project managers analyze potential risks.

R: Resources. Resources can be people, equipment, facilities, money, or anything else required for the completion of a project. Project managers are responsible for selecting the most efficient resources and delegating tasks appropriately.

T: Time. Project managers know the importance of time, so they create project timelines that include specific deadlines and communicate these expectations with the team. Effective time management is a valued trait in a project manager.
