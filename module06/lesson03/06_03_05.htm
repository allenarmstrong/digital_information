Work the Wizzy-Wig

Here is the cool part about using a WYSIWYG editor: As you type and format text, the WYSIWYG editor creates the HTML code for you. Give it a try! Follow the directions to see a WYSIWYG editor in action.


Directions

Use the blank WYSIWYG editor below and follow the steps:

  1. Type the following lines into the editor and fill in the blanks with your information:
    Hi! My name is __________.
    I enjoy the following:
    __________
    __________
  2. Highlight "Hi! My name is …" and select "H4.”
  3. Highlight your two likes and select the "1, 2, 3" icon to create a numbered list.
  4. Highlight your text for #1 and select "B" to bold.
  5. Highlight your text for #2 and select "I" to italicize.
  6. Select "Enter" twice to get to a new unnumbered line.
  7. Copy this image source link:
    http://i377.photobucket.com/albums/oo212/asda6389/th_smiling_sun.gif
  8. Select the "Insert Image" button (looks like a picture).
  9. Paste the image source link (http://i377.photobucket.com/albums/oo212/asda6389/th_smiling_sun.gif).
    When you paste in the image source, it will automatically insert the "http://." You must delete one set of "http://" in order to get the image to display properly.
  10. Select the source button (</>) on the toolbar to see the HTML coding for your content

Congratulations! You are officially a budding web designer. But, don't stop here. Go ahead and play with the WYSIWYG editor. Try adding your own text and images and play with the formatting options.
