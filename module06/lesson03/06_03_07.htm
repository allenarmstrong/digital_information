A Picture Is Worth a Thousand Clicks


John is off to a great start to his web page, but with text by itself, the page is just boring. Using images and graphics in a web page helps liven it up and make it more enjoyable for the reader. Since John is an architect, he clearly wants to include images related to his profession, such as blueprints, drafting tools, and completed projects. Adding an image to a web page follows a specific code:

<img src="images/image_name.jpg" alt="description of image content">

The code defines the folder where the image resides on the computer, images is the folder, and image_name is the file's name. The file extension, in this case jpg, defines the type of image. Finally, the alternate tag (alt) provides a textual description of the image for e-readers. E-readers will then describe the image to people who have this feature turned on in their browser.
