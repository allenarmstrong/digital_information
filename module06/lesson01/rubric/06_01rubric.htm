06.01 Building Blocks for Web Design Rubric

(30 points possible)

Criteria Excellent
15–11 points		 Needs Improvement
10–0 points		 Score Yourself
Web Page Finished web page has all of the following: a centered header; a background color; text; a numbered list (1-3) with bolding, italics, and a font color; and an image. Finished web page is missing some or all of the following: a centered header; a background color; text; a numbered list with bolding, italics, and a font color; and an image.  
Reflection All reflection questions are answered completely and meaningfully. All reflection questions are not answered, and/or questions are not answered completely and/or meaningfully.
