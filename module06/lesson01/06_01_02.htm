Learn the Language

Do you know what football, physics, texting, and web pages have in common? They all have their own language. For example, football has terms like "flea flicker" and "onside kick," while physics uses words like "nuclear fission" and "thermal capacity." To truly understand a subject, you must learn its vocabulary. Web design is no different. Read the descriptions of the different coding languages used to create a web page.

Cascading Style Sheets is a stylesheet language used to describe how elements of HTML should be displayed onscreen.
The authoring language developed to create web pages; it defines the structure and layout of a web document.
High-level web programming language often used for web applications like Android apps.
JavaScript is widely used to add functionality and interactivity to web pages. Note: It differs from Java.
Hypertext Preprocessor is an open source scripting language executed on the server that is free to download.
Structured Query Language is code written as standalone lines known as queries and is often used to create, read, update, or delete data in a database.
