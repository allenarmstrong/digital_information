Get in Order

Now that you have an understanding of HTML language and the elements of a web page, it's time to talk about how to order the HTML tags properly. Here is a basic outline for a web page that has a heading and a paragraph:

Example:

<!DOCTYPE html>
<html>
<head>
    <title>Page Title</title>
</head>
<body>
    <h1>Heading</h1>
    <p>Paragraph</p>
</body>
</html>
