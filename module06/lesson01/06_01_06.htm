Convert My Code

Ready to start designing a web page? Text editors can help you do it! There are free text editors available online that will convert your HTML code into an HTML web page. Give it a try!

  1. Copy and paste the following code into the blank text editor screen below: 
    <!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head> 
<body> 
<h1>You're a Web Designer!</h1> 
</body>
</html>
  2. Select the source button </> to view your results.
Use the HTML Cheat Sheet and the skills you have learned so far in this lesson to practice writing HTML code in the text editor.

Building Blocks

 

As you know, web pages can vary. But one thing is certain: even the best web designers had to start with the basics. To create a basic web page, follow these instructions:

  1. Sign in to your Office Online account.
  2. Open your OneDrive.
  3. Select "New."
  4. Select "Plain text document."
  5. Assign a file name.
    Your document will automatically be saved as a text (.txt) file.
  6. Enter your HTML code.
  7. Rename your file with the ending .html.
 

Your HTML document will be saved to your OneDrive. To see your document as an HTML web page, download your file. Your web page will open in your browser.
