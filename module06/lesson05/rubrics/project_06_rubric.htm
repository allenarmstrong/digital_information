06.05 Module Six Project Rubric

(100 points possible)

Criteria Excellent
10–7 points		 Needs Improvement
6–0 points		 Score Yourself
Storyboard All elements of the storyboard are completed. Storyboard is missing some or all of the elements.  
Web Page Components The web page contains all the components (title, heading, image, link, numbered list, footer, and content). The web page is missing some or all of the components (title, heading, image, link, numbered list, footer, and content).  
Web Page Colors A font color and background color are used. A font color and/or background color are not used.  
Web Page Layout Center, left, and right alignments are used. Center, left, and/or right alignments are not used.  
Web Page Emphasis Bolding and italics are used. Bolding and/or italics are not used.  
Color Theory The font and background colors support color theory principles and contrast effectively. The font and/or background colors do not support color theory principles and/or contrast effectively.  
Overall Design The overall design is visually appealing and the layout is organized effectively. The overall design is not visually appealing and/or the layout is not organized effectively.  
Content The content supports the purpose of the web page. The content does not support the purpose of the web page.  
Mechanics Proper spelling, grammar, and punctuation are used. Proper spelling, grammar, and/or punctuation are not used.  
File Management The files are saved correctly and a zipped folder is used. The files are not saved correctly and/or a zipped folder is not used.
