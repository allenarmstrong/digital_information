Oh So Special(ized)


Through his research, John comes across an article titled "Specialized Web-Design Software". He learns that specialized web-design software programs help create and incorporate more advanced items and additions for web pages. He reads about a few of them.


Adobe Flash is a multimedia and software platform that is used for the creation of animation, graphics, web applications, website-user interfaces, games, and rich Internet applications that can be viewed and played in Adobe Flash Player. Within Flash are Small Web Format (SWF) files.

Adobe Shockwave is a multimedia platform similar to Flash in that it is also used to add animation, games, and interactivity to web pages. It is different because the Shockwave browser plug-in is not available for Solaris or Linux operating systems, meaning Shockwave content won't be able to be displayed for users on those operating systems. Shockwave is also used for online learning applications.

Adobe Director is a multimedia application platform used to create web content, interactive games, CD-ROMS, and stand-alone kiosks. Applications made in Adobe Director are able to be published and viewed on the Internet for users who have the Shockwave plug-in installed.

Adobe Dreamweaver is the industry standard for creating HTML pages and full web sites. It offers a code view, a WYSIWYG view, and a split view to see code and display views side-by-side.

John really likes incorporating graphics into his web pages and the different ways to do so. He has latched onto web-based applications simply for their accessibility. Lastly, John has learned about different specialized web-design programs that he can use to make his website top notch. He feels confident and starts to look at web-hosting pricing. He is ready to start building his website.
